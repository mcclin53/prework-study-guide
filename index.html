Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The head element is not seen by the user
  • The body element represents the visible content shown to the user.
  • HTML attributes are always defined in the beginning, not the closing element
  • h1: largest heading h2:smaller h3:even smaller etc.
  • The title element is important for the Search Engine Optimization

CSS

  • Inline CSS uses the style attribute to change styling an elecment directly in the html file
  • Internal style sheet - contains CSS rules for the webpage in the head section of the HTML file
  • External CSS style sheet allows decelopers to keep the CSS rules in a separate file
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit: commits changed to the repository (-m "": adds a message with our commit)
  • git pull origin main: syncs the local branch with the remote branch in GitHub
  • git push origin main: pushes the changes to the main branch
